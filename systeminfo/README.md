systeminfo
===============================

version number: 0.0.1
author: Sophie Heseltine

Overview
--------

A python package that can be installed with pip.

Installation / Usage
--------------------

To install use pip:

    $ pip install systeminfo


Or clone the repo:

    $ git clone https://github.com/sophieheseltine/systeminfo.git
    $ python setup.py install
